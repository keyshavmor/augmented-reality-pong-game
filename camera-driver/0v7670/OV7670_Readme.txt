1. Clock on the 0v7670 folder
2. Double Click on the 0V7670.xpr file ( You need Vivado 2015.4 )
3. Run Synthesis Design as Project Opens
4. Run Implementation
5. Gnerate Bit Stream
6. Open Hardware Manager
7. Click on Auto Connect
8. Click on Open Target
9. Click on Program the Board
10. Make sure the ZedBoard is powered on and Jumpers are not altered.
11. Connect a VGA Screen/Projector Input to Board VGA Output
12. You should see an image of where the Camera is pointing
13. Adjust focus of the Camera by rolling the lens.
